# Predicția riscului de diabet folosind modele de Machine Learning

Acest proiect realizează preprocesarea datelor medicale și antrenarea mai multor modele de clasificare (Logistic Regression, Random Forest și Gradient Boosting) pentru estimarea riscului de diabet, incluzând și evaluarea performanței, calibrarea probabilităților și interpretarea rezultatelor.
